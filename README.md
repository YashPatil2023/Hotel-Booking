## Step 1: Install Java JDK

Make sure you have Java JDK installed on your machine. If not, download and install it from [here](https://www.oracle.com/java/technologies/javase-downloads.html).

## Step 2: Clone the Repository

Use the following command to clone the repository:

```bash
git clone [repository_url]
```

Navigate to the "Hotel Booking" folder.

## Step 3: Open Workbench

Open your favorite workbench to manage the database.

```sql
-- Set up the Database
create database Register;
use Register;

-- Create User Table
CREATE TABLE user (
    FM varchar(30),
    LM varchar(30),
    EM varchar(30),
    PW varchar(30),
    UN varchar(30)
);

-- Create Booking Table
CREATE TABLE Booking (
    UN VARCHAR(50),
    AD VARCHAR(50),
    MN VARCHAR(50),
    RT VARCHAR(50),
    FA VARCHAR(50),
    SD VARCHAR(50),
    NR VARCHAR(50),
    ID VARCHAR(50),
    OD VARCHAR(50),
    MA VARCHAR(50),
    FE VARCHAR(50),
    CH VARCHAR(50)
);
```
